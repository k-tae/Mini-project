배열과 포인터

배열명 + i (i는 배열의 인덱스 - 배열 요소의 자료형에 따라 메모리 상의 크기는 달라짐)

변수의 영역과 데이터 공유
(언어에서 사용영역(유효영역)에 따른 변수의 종류

1. 지역변수 (로컬변수)
{ } 내부에서만 유효한것, 함수의 내부에서만 사용하는 변수
초기화를 하지 않으면 쓰레기값

2. 전역변수 (글로벌변수)
{ } 외부에서 선언됨. 어디서든 접근 가능.
초기화를 하지 않으면 자동으로 0으로 초기화
전역변수는 초기화할 때 반드시 상수로 초기화

3. 정적변수 (static 변수)
static을 앞에 붙여서 선언
초기화는 단 한 번만 함.
초기화를 하지 않으면 자동으로 0으로 초기화
함수가 종료되어도 메모리 상에 남아 있음.
{ } 내에서만 유효함.

사용자 정의 자료형

1. 구조체(struct)
서로 다른 자료형을 모아 하나의 단일 자료형처럼 사용
모든 구조체 멤버 변수에 메모리에 할당

2. 공용체(union)
서로 다른 자료형을 모아 하나의 단일 자료형처럼 사용
공용체 멤버 변수가 메모리를 공동으로 사용
